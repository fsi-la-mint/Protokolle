# Probleme
- Hauptproblem: fehlendes Engagement und fehlende Verantwortung für Aufgaben
- Anlernen durch Ältere hat gefehlt bzw. war nur minimal vorhanden
- Studimails werden im Moment nicht zeitgemäß beantwortet
- IT-Infrastruktur ist sehr feindselig gegenüber Newcomern (besonders nicht-Informatikern)
    - insbesondere GitHub ist ein Problem (sehr verwirrend)


# Ideen
- Wiki für Anleitungen zur Organisation oder andere nützliche Dinge
- Proto über Mail statt nur über Git 
- Zuständigkeitsliste: wird jetzt in Sitzung erstellt
    - Fazit: ist eigentlich recht gut abgedeckt
- FSI-Wiki
- Sprechstunde (wöchentlich für 1h) mit Ankündigung auf Website, Studi-Mail-Verteiler

# Beschlüsse
- Mails werden per se von Mailverteiler-Verwaltung beantwortet
    - Studiengangsverantwortlichen beantworten Mails, die ihren eigenen Studiengang betreffen
    - Mails bezüglich Events werden von Eventverantwortlichen beantwortet
- Proto wird über Mail verschickt
    - Bei Kritik trotzdem noch über die Git-Kommentare die Anmerkungen
